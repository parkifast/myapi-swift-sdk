PlacesApiManager

public struct PlacesApiManager

Convenience structure for PlacesApi to easily access the suite of APIs.

  • search(lat:lon:radius:type:next:limit:onSuccess:onError:)

    Places API - Search: Searches around a set of Coordinates within a certain radius (in meters) to retrieve a list of nearby points of interest of a specific type (optionally).

    Declaration

    Swift

    public func search<SearchPlacesResponse: Codable>(
    lat: Double,
    lon: Double,
    radius: Int,
    type: String? = nil,
    next: String? = nil,
    limit: Int? = 25,
    onSuccess: @escaping DeserializedModelCompletion<SearchPlacesResponse>,
    onError: @escaping ErrorCompletion
)

    Parameters

    lat

    Latitude.
    lon

    Longitude.
    radius

    Radius of search in meters.
    type

    The specific type of point of interest (optional).
    next

    A next page results token (optional).
    limit

    The max number of results that should be retrieved (default 25, max 250) (optional).
    onSuccess

    A closure which is called with SearchPlacesResponse object.
    onError

    A closure which is called with a String representation of the error.

    Return Value

    A SearchPlacesResponse object
